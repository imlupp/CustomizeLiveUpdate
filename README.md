# 📦 CustomizeLiveUpdate – 取件码/取餐码实时通知小工具  

一个简洁实用的 Android 应用，帮你快速将 **取件码、取餐码** 转为 **实时活动通知**，方便在 **锁屏、灵动岛、AOD** 上随时查看✨  



## 🚀 当前功能亮点  
- 📝 输入取件点/取餐柜 + 取件码/取餐码 → 立即发送灵动岛通知  
- 📋 支持多个码同时存在，列表清晰展示  
- ✅ 点击 **“已取件/已取餐”** 按钮 → 自动删除通知及列表项，清爽无残留  
- 🍵 图标适配：为咖啡、奶茶、西餐、中餐等不同类型餐品匹配专属图标，直观好认  



## ⚠️ 使用须知  
**最低系统要求**：Android 16 (API 36)  
**适用机型示例**：  
- Xiaomi HyperOS 3.0.300+  
- ColorOS 16+  
- Google Pixel 及相关原生 Android 设备  

> 📌 注意：部分手机厂商可能对实时活动通知有额外限制，请确保系统支持 **Google Live Update** 并已开启相应功能开关。



## ❓ 常见问题  

**Q：通知不显示为灵动岛？**  
A：请按以下步骤检查：  
1️⃣ 确认手机系统为 **Android 16 或以上**  
2️⃣ 在应用设置中授予 **通知权限**  
3️⃣ 进入系统设置 → 找到并开启 **“实时动态”**（或类似名称）选项
