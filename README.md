# 📦 CustomizeLiveUpdate – 快递取件码实时通知小工具  

一个简洁实用的 Android 应用，帮你快速将快递取件信息转为 **实时活动通知**，方便在 **锁屏、灵动岛、AOD** 上随时查看取件码✨  

---

## 🚀 当前功能亮 
- 📝 输入取件点 + 取件码 → 立即发送灵动岛通知  
- 📋 支持多个取件码同时存在（列表清晰展示）  
- ✅ 点击 **“已取件”** 按钮 → 自动删除通知及列表项，清爽无残留  

---

## ⚠️ 使用须知  
**最低系统要求**：Android 16 (API 36)  
例如：  
- Xiaomi HyperOS 3.0.300  
- ColorOS 16  
及更高版本  

> 📌 原始设备制造商 (OEM) 可能对实时活动通知有额外限制，请确保系统支持 Google Live Update 并已开启相应开关。

---

## ❓ 常见问题  

**Q：通知不显示为灵动岛？**  
A：请按以下步骤排查：  
1️⃣ 确认手机系统为 **Android 16 或以上**  
2️⃣ 确保应用已获取 **通知权限**
3️⃣ 进入系统设置 → 开启 **“实时动态”**（或类似名称）选项
